Junk Cleaner

Простой чистильщик мусора для Windows. Удаляет временные файлы, кэши браузеров, корзину и другой безопасный мусор. Системные файлы и личные документы не трогает.

Скачать .exe (11 МБ) GitHub Все версии

Что чистит

  • Временные файлы (%TEMP%)
  • Системные временные файлы
  • Кэш Prefetch и обновлений Windows
  • Кэш миниатюр
  • Отчёты об ошибках
  • Кэш Chrome, Edge, Firefox
  • Корзину
  • Кэш DNS

Что не трогает

  • Документы, Загрузки, Рабочий стол
  • Установленные программы
  • Windows\System32
  • Пароли и закладки браузеров

Безопасность

  • Не собирает личные данные
  • Не работает в фоне
  • Не добавляет автозапуск
  • Удаляется простым удалением .exe
  • Занятые файлы пропускаются

Как пользоваться

  • Скачайте и запустите .exe
  • Дождитесь сканирования
  • Выберите категории
  • Нажмите «Очистить выбранное»
  • Для системных категорий — запуск от администратора

English

Junk Cleaner is a free portable Windows utility for safe cleanup of temp files, browser caches, thumbnails, and recycle bin. No installer, no ads, open source.
