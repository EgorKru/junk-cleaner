## 1. itch.io (рекомендуется, бесплатно)

1. Зарегистрируйся: https://itch.io/register
2. Create new project → Kind: **Tool**
3. Заполни из `docs/itch-page.html`
4. Upload file или укажи external link на GitHub
5. Visibility: **Public**
6. Запусти `scripts\publish_itch.bat` для автозагрузки (после `butler login`)

## 2. AlternativeTo

1. Открой: https://alternativeto.net/software/ccleaner/
2. Нажми **Suggest alternative** / **Add application**
3. Заполни:
   - **Name:** Junk Cleaner
   - **Website:** https://egorkru.github.io/junk-cleaner/
   - **License:** Free / Open Source
   - **Platform:** Windows
   - **Description:** (из LAUNCH_POSTS.md)
