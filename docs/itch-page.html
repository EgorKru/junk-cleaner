Junk Cleaner

Free portable junk cleaner for Windows. Removes temp files, browser caches, thumbnails, crash dumps, and empties the recycle bin.

Safe: does not touch Documents, Downloads, Desktop, or system files.

No installer. Just download and run.

Features

  • User & system temp files
  • Browser caches (Chrome, Edge, Firefox)
  • Thumbnail cache
  • Crash dumps
  • Recycle bin
  • DNS cache flush
