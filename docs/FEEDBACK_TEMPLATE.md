**Тема:** Помоги протестировать бесплатный чистильщик для Windows (5 минут)

Привет! Я сделал простую программу **Junk Cleaner** — она удаляет временный мусор с компьютера (кэши, temp-файлы, корзину). Нужна помощь с тестом перед публикацией.

**Скачать:** https://github.com/EgorKru/junk-cleaner/releases/download/v1.0.0/JunkCleaner-1.0.0-windows.exe

**Что проверить (5 минут):**
1. Запускается ли программа?
2. Понятен ли интерфейс?
3. Сканирование показывает размеры?
4. Очистка одной категории (например, DNS) работает без ошибок?
5. Windows не ругается антивирусом?

**Ответь, пожалуйста:**
- ОС: Windows 10 / 11
- Запустилось: да / нет
- Понятно ли: да / нет / что непонятно
- Ошибки: были / не были (какие)
- Что добавить: ...
- Оценка 1–10: ...

Спасибо!
