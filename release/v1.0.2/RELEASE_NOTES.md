### Что это
Бесплатный portable чистильщик мусора для Windows. Удаляет временные файлы, кэши браузеров, корзину. Безопасно для системы.

### Что нового в 1.0.2
- Красивая иконка приложения
- Улучшенный интерфейс
- Открытый исходный код (MIT)
